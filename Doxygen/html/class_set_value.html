Detailed Description

This compute class can be used to set the value of a cell.

This compute class together with BoxLoops::loop(...) can be used to set the values in a Box or in a LevelData to a constant value. Note: The functionality is the same as GRLevelData.setVal but this compute class makes it possible to bundle calculations up in a ComputeClassPack to avoid doing several loops.
