Detailed Description

template<class matter_t>
class MatterWeyl4< matter_t >

Calculates the Weyl4 scalar for spacetimes with matter content.

This class calculates the Weyl4 scalar real and im parts. It inherits from the Weyl4 class and adds in the matter terms as appropriate depending on the formulation
