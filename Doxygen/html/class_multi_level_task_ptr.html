Detailed Description

template<class level_t = GRAMRLevel>
class MultiLevelTaskPtr< level_t >

This is just an interface for the AMR scheduler to call some level_t function on every AMRLevel This can either be called directly by calling execute, or passed to an AMR (as GRAMR) by doing gr_amr.schedule(me) (this version will make it be called only after plot files are written, if that is ever an interest)
