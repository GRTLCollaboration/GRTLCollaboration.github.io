Detailed Description

template<class potential_t = DefaultPotential>
class ScalarField< potential_t >

Calculates the matter type specific elements such as the EMTensor and.

This class is an example of a matter_t object which calculates the matter type specific elements for the RHS update and the evaluation of the constraints. This includes the Energy Momentum Tensor, and the matter evolution terms. In this case, a scalar field, the matter elements are phi and (minus) its conjugate momentum, Pi. It is templated over a potential function potential_t which the user must specify in a class, although a default is provided which sets dVdphi and V_of_phi to zero. It assumes minimal coupling of the field to gravity.
