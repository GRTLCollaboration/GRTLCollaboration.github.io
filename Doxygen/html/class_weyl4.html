Detailed Description

Calculates the Weyl4 scalar for spacetimes without matter content.

This class calculates the Weyl4 scalar real and im parts using definitions from Alcubierres book "Introduction to 3+1 Numerical Relativity". We use a decomposition of the Weyl tensor in electric and magnetic parts, which then is used together with the tetrads defined in "gr-qc/0104063" to calculate the Weyl4 scalar.
