Detailed Description

template<class data_t>
class Cell< data_t >

Encapsulates information about the position of a cell.

It contains the position of the cell on the Chombo grid and the index of the flattened Chombo array where the data is stored (both for the input and the output data of the FABDriver). The main use of Cell is to hand information on the current cells from the FABDriver to the compute classes.
