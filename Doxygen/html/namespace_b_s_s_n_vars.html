Detailed Description

Namespace for BSSN vars.

The structs in this namespace collect all the BSSN variables. It's main use is to make a local, nicely laid-out, copy of the BSSN variables for the current grid cell (Otherwise, this data would only exist on the grid in the huge, flattened Chombo array).
