Detailed Description

template<class level_t = GRAMRLevel>
class MultiLevelTask< level_t >

This is just an interface for the AMR scheduler to call some GRAMRLevel (or any other Example specific level) function on every AMRLevel Satisfies syntax of Chombo's Scheduler such that it can be passed to GRAMR and be scheduled.
