Detailed Description

template<bool is_out_data>
struct CellIndex< is_out_data >

Represents an index in the flattened Chombo array.

CellIndex only stores one integer (the position in the flattened Chombo array of the cell the CellIndex represents). A CellIndex expects a bool as template arugment to check whether it represent "in" or "out" data of the FABDriver. For better readibility use CellIndexIn and CellIndexOut instead of CellIndex<false> and CellIndex<true> respectively.

Template Parameters
is_out_dataA bool that makes sure that the indices in the "in" and "out" arrays are never confused.
