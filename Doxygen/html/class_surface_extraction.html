This integrate function can be used if you only want to integrate one integrand.

Integrate some integrand dependent on the interpolated data over the surface.

It calls add_integrand() and integrate() The last argument is whether to broadcast the result to all MPI ranks or just keep on rank 0. Most use cases won't need this set to true.

The integrand function should be of the signature double integrand(std::vector<double> data_here, double a_surface_param_value, double a_u, double a_v) where data_here is a vector of all the interpolated variables at the point specified by the other arguments.
